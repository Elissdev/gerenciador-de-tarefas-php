# Gerenciador de Tarefas Full Stack 

Um sistema completo de gerenciamento de tarefas desenvolvido com foco em qualidade de software, conteinerização e interface moderna.

## Tecnologias Utilizadas
* **Frontend:** HTML, JavaScript e TailwindCSS
* **Backend:** PHP (API REST)
* **Banco de Dados:** MySQL
* **Infraestrutura:** Docker e Docker Compose

## Como rodar o projeto

1. Certifique-se de ter o Docker e o Docker Compose instalados.
2. Clone este repositório.
3. Abra o terminal na pasta do projeto e rode o comando:
   `docker-compose up -d`
4. O banco de dados será inicializado automaticamente. Acesse a interface em:
   `http://127.0.0.1:5500/frontend/index.html`
